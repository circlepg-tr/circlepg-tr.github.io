# 陶酔ロマンへようこそ

東方同人サークル『**陶酔ロマン**』のオフィシャルサイトへようこそ！

このサイトでは、サークルの活動情報やメンバーの作品を掲載しています。

## 活動内容

[**八雲 櫻**](https://x.com/yakumoougama?s=21&t=HgKqcnVyxch9ihDE10UdoA) と [**すためりあ**](https://x.com/rein_13424?s=21&t=HgKqcnVyxch9ihDE10UdoA) の2名で、東方Projectの二次創作を中心に不定期で活動を行う予定です。

現在はサークルとしての活動は特に行なっておりませんが、活動に関する最新情報は各メンバーのSNSやこのサイトでお知らせする予定です。

それまでの間は、ブログを書いたり個人の制作物をBoothにて販売しております。気になる方はぜひプロフィールのアイコンからチェックしてみてください。
